
### Format of Client Info File
```
1 line per client
userid|password|ip|port|isLoggedIn
```

### Format of Group Info File
```
5 lines per group
groupid
owner-userid
group-members-userid's
pending-request-userid's
list-of-filenames
```

### Format of File Info File
```
6 lines per file
groupid|fileName
fileName
fileSize
fileSHA1
seederList
leecherList

```
